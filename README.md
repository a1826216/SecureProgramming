JSON (https://github.com/nlohmann/json)

## Setup Instructions
Install libraries (for Ubuntu/WSL):

```
sudo apt update
sudo apt install nlohmann-json3-dev 
sudo apt install libwebsocketpp-dev
```

OpenSSL should already be installed on most systems.
